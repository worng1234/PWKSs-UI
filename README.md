# ติดตั้งระบบ

เตรียมระบบก่อนที่จะรันตามขั้นตอนนี้

```bash
1. ติดตั้ง nodejs v16
2. เข้าสู่โฟลเดอร์ระบบแล้วพิมพ์คำสั่ง npm i
3. รันระบบพิมพ์คำสั่ง npm run dev
```

# ระบบห้องเรียน
